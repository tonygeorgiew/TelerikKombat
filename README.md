#Telerik Kombat
Combat multiplayer game. 
Main purpose of the game: Collect more points and kill your opponent!

Player one controls :

Moves:
Up - kye W
Down - key S
Left - key A
Right - kye D

Fight - key F

Player two controls:

Moves:
Up - arrow key Up
Down - arrow key Down
Left - arrow key Left
Right - arrow key Right

Fight - key M
